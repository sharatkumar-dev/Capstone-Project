# Reusable Agent Prompts

This folder stores the system instructions and operational guides for our three specialized tax compliance agents:

- **`prompts/router_agent.txt`**: Guidelines for taxpayer classification (Section 44AD vs. Section 44ADA) and onboarding parameters validation.
- **`prompts/vision_agent.txt`**: Guidelines for parsing invoice images/PDFs and extracting line items, payment modes, and GST.
- **`prompts/calculator_agent.txt`**: Guidelines for applying mathematical presumptive formulas, checking statutory limits, and formatting tables.
